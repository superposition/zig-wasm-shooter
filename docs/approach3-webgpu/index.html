Controls

Use Arrow Keys or WASD to move your ship (cyan triangle).

  • Dodge orange triangular enemies
  • Dodge red rectangular obstacles
  • Stay within the gray hallway
  • +1 point for each enemy dodged
  • +2 points for each obstacle dodged
  • -5 points for hitting an enemy
  • -10 points for hitting an obstacle
