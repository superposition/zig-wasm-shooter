Zig WASM Shooter

A vertical shooter game in Zig + WebAssembly with three rendering approaches

Canvas2D Easy

JavaScript renders the game using Canvas2D API. Zig handles game logic only. Simple and works everywhere.

Canvas2D All Browsers

WebGL Medium

True GPU rendering via WebGL 1.0. Zig calls WebGL functions for rendering with shader support.

WebGL 1.0 GLSL Shaders All Browsers

WebGPU Advanced

Modern GPU API with best performance. Uses WGSL shaders and cutting-edge browser features.

WebGPU WGSL Shaders Chrome/Edge 113+

Controls

W / Arrow Up - Move up

A / Arrow Left - Move left

S / Arrow Down - Move down

D / Arrow Right - Move right

Space - Shoot

Objective: Avoid red enemies and gray obstacles to survive!

Built with Zig + WebAssembly | Source on GitHub
